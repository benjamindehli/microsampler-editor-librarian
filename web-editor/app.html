microSAMPLER EDITOR — HELP

Pads — click to edit. Bright LED = sample loaded (instant audition); dim LED = used but not loaded yet. Hold ▶ (lower-right) to play the sample on the device. Drag a pad onto another to copy / swap; drop a .wav onto a pad to load it.

Waveform — drag the S/E edges to set start/end (with ZERO-SNAP on, a dragged point snaps to the nearest zero crossing so the cut doesn't click; type a value for an exact frame). Scroll to zoom, drag the body to pan, and double-click (or FIT) to see the whole sample.

RECEIVE re-reads the bank from the device — use it after editing on the hardware (panel edits only reach the app while the device is on its sample-edit page). The app also auto-receives when you refocus the window.

Pattern editor — on the PATTERNS view, ✎ EDIT opens a piano roll: click to add a note, drag to move it, drag its right edge to resize, scroll over it for velocity, Del to remove. Switch the SMP (pad) / KBD (pitched) track, set bars and the keyboard track's sample, then SAVE TO DEVICE. Works on an empty pattern to build one from scratch.

RAM vs flash — all edits/transfers target the device's CURRENT bank (RAM); they're lost on power-off. Save on the device, or restore a backup to a USER bank (UTILITY) to persist.

Memory meter — exact: every sample is downloaded on connect (you'll see a progress bar), so the readout is real and waveforms are instant.

Panic (⏻, top-right) — all-sound-off + stop, if a note ever sticks.

Keyboard (below the editor) — an on-screen piano mirrors the 36 pads; click any key to play it through the device (real envelope/FX). Tick ⌨ TYPE TO PLAY to play from your computer keyboard too: home row A S D F G H J K = white keys, top row W E · T Y U = black keys (the piano shows which key is which), and Z / X — or the ◀ ▶ buttons — shift the octave (C3/C4/C5). While it's off, those keys stay free for the shortcuts below.

Shortcuts — ← → ↑ ↓ move pad · Space audition · + − 0 waveform zoom · R receive · ⌘/Ctrl+Z undo · ⌘/Ctrl+⇧Z redo · ? this help.
